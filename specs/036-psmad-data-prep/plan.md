## Implementation Design

### Phase 1: Extend feature_extractors.py

Add two new functions to `backend/ml_data/utils/feature_extractors.py`:

#### `extract_fft_features_single_axis(window, sampling_rate_hz, low_hz, high_hz)`
```python
def extract_fft_features_single_axis(
    window: np.ndarray,
    sampling_rate_hz: float = 37.0,
    low_hz: float = 3.0,
    high_hz: float = 12.0
) -> Dict[str, float]:
    """
    Extract FFT-based tremor-band features for a single axis.
    Returns dict with keys: 'dominant_freq', 'tremor_energy'
    """
    N = len(window)
    fft_vals = np.fft.rfft(window)
    fft_magnitude = np.abs(fft_vals)
    freqs = np.fft.rfftfreq(N, d=1.0 / sampling_rate_hz)
    
    band_mask = (freqs >= low_hz) & (freqs <= high_hz)
    band_freqs = freqs[band_mask]
    band_magnitudes = fft_magnitude[band_mask]
    
    if len(band_magnitudes) == 0:
        return {'dominant_freq': 0.0, 'tremor_energy': 0.0}
    
    tremor_energy = float(np.sum(band_magnitudes ** 2))
    dominant_freq = float(band_freqs[np.argmax(band_magnitudes)])
    
    return {'dominant_freq': dominant_freq, 'tremor_energy': tremor_energy}
```

#### `extract_fft_features_all_axes(window, axis_names, sampling_rate_hz, low_hz, high_hz)`
Mirrors the existing `extract_features_all_axes()` pattern — loops over axes and prefixes keys as `dominant_freq_{axis}`, `tremor_energy_{axis}`.

### Phase 2: Create 4_psmad_pipeline.py

The script follows this execution flow:

```
1. Parse CLI args (paths, window_size=100)
2. Load metadata from AdditionalData.xlsx
3. Discover files in both group folders
4. Filter out files ending in *00.csv (validation tests)
5. For each valid recording:
   a. Load CSV with pandas
   b. Rename columns: T→Timestamp, AX→aX, AY→aY, AZ→aZ, GX→gX, GY→gY, GZ→gZ
   c. Compute sampling_rate from median(diff(Timestamp)) in Hz
   d. If len(df) < window_size: warn and skip
   e. Create non-overlapping windows: stride=window_size=100
   f. For each window:
      - Extract time-domain features: extract_features_all_axes(window, AXIS_NAMES)
      - Extract FFT features: extract_fft_features_all_axes(window, AXIS_NAMES, sampling_rate)
      - Combine into one dict, add label
      - Append to rows list
6. Build DataFrame from rows list
7. Validate: assert no NaN, no Inf, label in {0,1}
8. Save to ml_data/processed/ready_for_training_features.csv
9. Print summary report
```
